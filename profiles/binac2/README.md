# BinAC 2

The bwForCluster BinAC 2 supports researchers from the broader fields of Bioinformatics, Astrophysics, Geosciences, Pharmacy, and Medical Informatics in Baden-Württemberg.

For general information about BinAC 2, including hardware specifications,
available software, and usage policies, please refer to the official wiki:

> https://wiki.bwhpc.de/e/BinAC2
